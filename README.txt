Instructions for use:
To use as a general purpose search engine, set the seed address in SearchEngineSpider.aspx
To use as a static site search engine, set the base address in SearchEngineCrawler.aspx and change the Page_Load function in SearchEngine.aspx to redirect to SearchEngineCrawler.aspx instead of SearchEngineSpider.aspx
